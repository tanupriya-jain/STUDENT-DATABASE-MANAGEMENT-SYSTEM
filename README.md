# Student Database Management System

This repository contains an SQL project designed for managing student data. It includes an SQL script that sets up a simple student information management system.

## What's Inside
A `.sql` file with table creation statements, sample data inserts, and a few example queries.

## How to Use
1. Import the `.sql` file into your SQL database of choice (for example, MySQL, PostgreSQL).
2. Run the script to create the tables and insert the sample data.
3. Use the provided queries or build your own to manage and explore the student information.

## Prerequisites
 A working SQL database environment (like MySQL or PostgreSQL).
  DBeaver or any other SQL client if you want to modify or extend the script.
